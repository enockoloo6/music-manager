# 🎹 Universal Keyboard Song & Style Manager

A digital songbook designed for keyboardists to store and retrieve specific keyboard settings (Styles, Tempo, Key, and Menu Location) across multiple hardware models like the **Yamaha PSR-S670** and **PSR-S650**.

## 🚀 The Problem it Solves
Keyboardists often forget which specific rhythm or "Style" they used for a song, especially when moving between different piano models. This app acts as a "second brain," mapping your repertoire to specific keyboard settings so you can perform without searching through menus.

## 🛠️ Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Supabase (PostgreSQL & Auth)
- **Deployment:** Netlify
- **Routing:** React Router DOM

## 📋 Prerequisites
- Node.js installed
- A Supabase account
- A Netlify account

## ⚙️ Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <your-github-repo-url>
   cd music-manager
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run Locally:**
   ```bash
   npm run dev -- --port 5173 --strictPort
   ```

## 🗄️ Database Schema (SQL)
Run this in your Supabase SQL Editor to set up the tables:

```sql
CREATE TABLE keyboards (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  model_name TEXT NOT NULL UNIQUE
);

CREATE TABLE songs (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  song_name TEXT NOT NULL UNIQUE
);

CREATE TABLE styles (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  song_id BIGINT REFERENCES songs(id) ON DELETE CASCADE,
  keyboard_id BIGINT REFERENCES keyboards(id),
  beat_name TEXT NOT NULL,
  tempo INT,
  musical_key TEXT,
  keyboard_location TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Initial Data
INSERT INTO keyboards (model_name) VALUES ('Yamaha PSR-S670'), ('Yamaha PSR-S650');
```

## 📦 Key Dependencies
If you are reinstalling manually, these are the core libraries used:
- `@supabase/supabase-js`: To connect to the database.
- `react-router-dom`: For page navigation.
